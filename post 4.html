Atlas Tool to Display Demographic Information

Utilizing the atlas tool as well as rules-based symbology in QGIS

Posted by Anya Dougherty on November 27, 2023

The Prompt

The goal of this project was to display the migrational demographics of Germany as a whole but also the indivudal counties. This was accomplished by using the atlas feature and utilizing a Rule-Based style rendere to allow the viewer to only see the selected county.

Results

As we got further and further away, it [the Earth] diminished in size. Finally it shrank to the size of a marble, the most beautiful you can imagine. That beautiful, warm, living object looked so fragile, so delicate, that if you touched it with a finger it would crumble and fall apart. Seeing this has to change a man.
